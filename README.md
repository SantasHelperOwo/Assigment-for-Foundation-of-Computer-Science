# Foundation of Computer Science – Coursework (ST4015CMD)

## Overview
This repository contains all practical work, scripts, diagrams, and screenshots for the case study report:  
**Investigation and Analysis of Computing Data for Data Management**  
Softwarica College of IT & E-Commerce (in collaboration with Coventry University).

The coursework explores four major tasks:

1. **Task 1 – Encoding Formats & Secure Data Exchange**
   - Analysis of Base64, ASCII, and URL encoding.
   - Integration with HTTPS, TLS, and SMTP.
   - Risks of encoding-based obfuscation and proposed improvements.
   - Includes Python scripts and diagrams.

2. **Task 2 – Classroom Seating Arrangement (P vs NP Problem)**
   - Explanation of P vs NP complexity.
   - Brute force vs heuristic approaches.
   - Time complexity analysis.
   - Includes Python scripts demonstrating seating arrangements.

3. **Task 3 – College Club Membership Management**
   - Identification of data problems in unnormalized tables.
   - Normalization to 1NF, 2NF, and 3NF.
   - ER diagram creation.
   - SQL operations (INSERT, SELECT, JOIN).
   - Reflection on normalization and JOIN necessity.
   - Includes SQL schema, queries, and screenshots.

4. **Task 4 – References & Academic Integrity**
   - Harvard-style references.
   - Proper formatting and examiner-friendly presentation.
